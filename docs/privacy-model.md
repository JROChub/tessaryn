# Privacy Model

Capture is private by default. Raw input, shareable derivatives, publication,
indexing, retention, witness access, and location precision are separate policy
decisions.

The synthetic Origin demonstrates the minimum disclosure rule with a restricted
interior. The public fixture contains only an authorization boundary. It does
not contain hidden geometry, a blurred substitute, or a reversible residual.

Production capture remains blocked until it provides visible recording state,
local redaction, exclusion volumes, separate raw and public products, retention
controls, and adversarial residual-recovery tests.
