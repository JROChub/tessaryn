# Privacy Model

Capture is private by default. Raw input, shareable derivatives, publication,
indexing, retention, witness access, and location precision are separate policy
decisions.

The reference Origin demonstrates the minimum disclosure rule with a restricted
interior. The public fixture contains only an authorization boundary. It does
not contain hidden geometry, a blurred substitute, or a reversible residual.

The capture pipeline enforces operator confirmation, visible-recording intent,
local processing, identity-bound pixel masks, exclusion volumes, separate raw
and public products, and raw-frame omission. Restricted Loci use authenticated
multi-recipient encryption and signed replay-protected synchronization.

These controls do not replace a site-specific privacy review. A physical pilot
must still test aggregate leakage, residual channels, location metadata,
retention, bystander consent, and endpoint compromise before publication.
