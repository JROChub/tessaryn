# Visual Semantics v0

The native viewer begins at an Origin and condenses supported Cells outward.

- Unobserved space is open structure, not blurred filler.
- Restricted space is an authorization boundary with absent protected geometry.
- Disputed structures separate into visible alternatives.
- Semantic-only content is removable and independently bound.
- Brightness does not mean truth.
- Chronofold shares unchanged structure and separates changing temporal Cells.
- Scale Breathing keeps a selected object or relation as the perceptual anchor.

Map pins, panorama bubbles, road overlays, permanent minimaps, globe-first
navigation, and decorative proof particles are not native interaction language.
