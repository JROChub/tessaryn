# v0.22.0 Metric World Cell

This release replaces luminance-derived proxy depth with multi-view correspondence tracking, geometric triangulation, uncertainty-aware surfel fusion, explicit scale calibration, quality instrumentation, temporal Moments, and verified cross-device Metric World Cell reconstruction.

Release proof is enforced by `.github/workflows/ci-v022.yml` and requires:

- `VERSION` equals `0.22.0`
- the browser runtime emits `version: 22` World Cells
- `metric-reconstruction.ts` is present
- viewer tests pass
- the production viewer builds
- the production `world-cell-theater.html` route is emitted
- the release proof artifact is uploaded under a run-unique name
